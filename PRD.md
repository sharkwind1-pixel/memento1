# 메멘토애니 PRD (Product Requirements Document)

> Ralph Loop용 기능 명세서
> 각 기능 완료 시 체크하고 커밋

---

## 프로젝트 개요

**서비스명**: 메멘토애니 (Memento Ani)
**컨셉**: 반려동물 라이프 전체를 아우르는 따뜻한 커뮤니티 플랫폼
**디자인 감성**: My Little Puppy 게임 스타일, 파스텔톤, 밝고 따뜻한 천국 느낌
**타겟**: 반려인 전체 (시니어 포함 → 심플한 UI 필수)

---

## 핵심 기능 목록

### Phase 1: MVP 핵심 (우선순위 높음)

- [ ] **1.1 도트 미니미 시스템**
  - 가입자가 등록한 반려동물 정보 기반으로 도트 캐릭터 자동 생성
  - 싸이월드 미니미처럼 프로필에 표시
  - 종류(강아지/고양이/기타), 색상, 특징 반영
  - 추모 상태일 경우 날개/후광 등 천국 버전

- [x] **1.2 포인트 시스템 (FM코리아 스타일)**
  - 활동 포인트 적립: 글쓰기, 댓글, 좋아요, 출석 등
  - 포인트 사용처: 도트 미니미 꾸미기, 특별 이모티콘 등
  - 랭킹 시스템 (선택)
  - DB 테이블: point_transactions, profiles.points

- [ ] **1.3 AI 펫톡 개선**
  - 건강 기록 연동 → AI가 피드백 제공
  - "오늘 산책 30분 했어" → AI가 칭찬 + 건강 팁
  - 감정 트래킹 → 힐링 리포트 (주간/월간)

- [ ] **1.4 건강 기록 기능**
  - 백신, 구충제, 건강검진 기록
  - 리마인더 알림 (선택)
  - AI 펫톡과 연계

### Phase 2: 수익화 기능

- [ ] **2.1 지역 기반 광고 배너 시스템**
  - 사용자 지역 동의 후 위치 정보 수집
  - 해당 지역 근처 광고주 배너 노출:
    - 동물병원
    - 펫 헤어샵/미용
    - 애견샵/용품점
    - 펫 카페
    - 장례식장/추모 서비스
  - 광고주 관리 페이지 (어드민)
  - 광고 클릭/노출 통계

- [ ] **2.2 B2B 제휴 시스템**
  - 동물병원/장례식장 제휴 페이지
  - QR코드 → 추모페이지 자동 생성
  - 월 구독료 모델

- [ ] **2.3 프리미엄 구독**
  - 월 4,900원
  - AI 펫톡 무제한
  - 프리미엄 도트 미니미 아이템
  - 광고 제거

### Phase 3: 커뮤니티 강화

- [ ] **3.1 공동 추모 페이지**
  - 개별 반려동물 추모 페이지에 방명록
  - 가상 헌화/장난감 남기기
  - 가족/친구 초대 기능

- [ ] **3.2 SNS 요소 (점진적)**
  - 팔로우/팔로워 시스템
  - 홈 피드 (팔로우한 사람 글)
  - 인기 게시물 노출

## 기술 스택

- **Frontend**: Next.js 14, React, TypeScript, Tailwind CSS
- **Backend**: Supabase (PostgreSQL, Auth, Storage, Realtime)
- **AI**: OpenAI GPT-4 API
- **Deployment**: Vercel
- **Payment**: 포트원 (추후)
