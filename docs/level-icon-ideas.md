**고양이 특유의 표현 요소:**
- 고양이 특유의 세로 동공 (레벨이 올라갈수록 눈이 더 빛나게)
- 수염 표현 (가늘고 우아하게)
- 귀 안쪽 핑크색 터치
- 앞발 동그랗게 모은 자세 ("고양이 빵" 포즈) -- 특히 Lv.3

## 3. Other Pet 아이콘 세트 기획

### 3-1. 동물 종류 선정 전략

"other" 카테고리는 단일 종이 아닌 **다양한 소동물을 레벨별로 배치**하는 전략을 추천한다. 이렇게 하면:
- 다양한 반려동물 보호자가 자신의 동물을 발견하는 재미
- 레벨별로 확연히 다른 시각적 차이 (같은 종의 품종 차이보다 명확)
- "다음 레벨 동물은 뭘까?" 하는 호기심 유발

### 3-2. 레벨별 동물 선정

| 레벨 | 동물 | 선정 이유 | 표정/포즈 | 배경색 | 장식 |
|------|------|----------|----------|--------|------|
| **Lv.1** | **햄스터 (Hamster)** | 작고 동글동글, 초보 반려인 첫 동물로 친숙. 골든 햄스터 또는 펄 화이트 추천. 한국 반려동물 시장에서 인기 | 양 볼 가득 해바라기씨 물고 있는 모습 | 연회색 (#E5E5E5) | 없음 |

### 3-4. Other 아이콘 디자인 노트

**핵심 원칙: "다양하지만 통일감"**
- 7종 모두 다른 동물이지만, 동일한 수채화 치비 스타일로 통일
- 배경 원형 프레임 + 볼 터치(rosy cheeks)는 모든 동물에 적용
- 각 동물의 가장 매력적인 특징을 강조 (햄스터의 볼주머니, 토끼의 늘어진 귀, 고슴도치의 바늘 등)

**체형 차이 처리:**
- 작은 동물(햄스터, 고슴도치): 원형 프레임 안에 전신이 들어갈 수 있음
- 중간 동물(토끼, 친칠라): 상반신 중심
- 날개/비막 동물(앵무새, 슈가글라이더): 특수 부위가 프레임 밖으로 살짝 튀어나오는 연출 가능
- 거북이: 등딱지 무늬가 잘 보이도록 약간 위에서 내려다보는 각도

## 6. 한국 반려동물 앱 디자인 트렌드 분석

### 6-1. 주요 참고 서비스 레벨/뱃지 시스템

**당근마켓 (Karrot) - 매너온도 시스템**
- 36.5도 기본 시작, 활동에 따라 올라감
- 온도별 색상 변화 (파랑 -> 초록 -> 주황 -> 빨강)
- 단순하지만 직관적인 시각 피드백
- 참고: 숫자 기반이라 즉각적 이해가 쉬움

**배달의민족 (Baemin) - 등급 시스템**
- 비기너 -> 실버 -> 골드 -> 플래티넘 -> 다이아
- 전통적인 게임 등급 체계 차용
- 아이콘 + 컬러 조합으로 등급 시각화

**포잉 (Fohing) - 반려동물 SNS**
- 활동 기반 뱃지 시스템
- 산책 횟수, 게시물 수 등 활동별 뱃지
- 동물 종류별 커스텀 프로필 프레임

**카카오 같이가치 - 봉사활동 뱃지**
- 분야별 특화 뱃지
- 컬렉션 형태의 뱃지 모음
- 활동 마일스톤별 잠금 해제

**네이버 카페 등급 시스템**
- 가입인사 -> 새싹 -> 파란잎 -> 우수 -> 정회원 등
- 식물 성장 메타포가 일반적
- 커뮤니티 참여도 기반

### 6-2. 2025-2026 한국 반려동물 앱 디자인 트렌드

**1. 감성적 수채화/파스텔 UI**
- 강아지풀, 하늘재, 펫노트 등 한국 펫앱들이 파스텔 톤 UI 채택
- 하드한 플랫 디자인에서 소프트하고 둥근 디자인으로 전환
- 손그림 느낌의 일러스트 요소 선호

**2. 동물 종류별 맞춤 경험**
- 강아지/고양이뿐 아니라 소동물, 파충류까지 지원 범위 확대
- 종류별 맞춤 아이콘, 맞춤 테마 제공이 차별화 요소
- "내 아이만의" 개인화 경험 중시

**3. 게이미피케이션 요소 강화**
- 단순 포인트 -> 레벨 시스템 + 뱃지 컬렉션 + 업적
- 시각적 보상(아이콘 변화)이 텍스트 보상보다 효과적
- "다음 레벨"에 대한 기대감 조성이 리텐션 핵심

**4. 커뮤니티 내 신뢰 지표**
- 당근마켓 매너온도처럼, 레벨이 신뢰의 지표 역할
- 높은 레벨 = 오래 활동한 신뢰할 수 있는 회원
- 입양/분실동물 게시판에서 특히 중요

**5. 메모리얼/감성 콘텐츠**
- 무지개다리 테마의 추모 공간 증가
- 따뜻하고 치유적인 디자인 언어
- 슬프지 않은 이별, 감사의 기록 강조

## 7. 레벨 아이콘 추가 연출 아이디어

### 7-1. 배경 이펙트 차별화

현재 코드(`LevelBadge.tsx`)에서 `hasSparkle`과 `hasGlow` 속성을 사용하고 있다. 추가로 고려할 수 있는 CSS 기반 이펙트:

| 레벨 | 이펙트 | 구현 방법 |
|------|--------|----------|
| Lv.1-3 | 정적 아이콘 | 이펙트 없음 (심플 표시) |
| Lv.4 | 미묘한 그림자 | `box-shadow: 0 2px 8px rgba(0,0,0,0.1)` |
| Lv.5 | 반짝이 도트 | 현재 구현된 sparkle dot (골드) |
| Lv.6 | 반짝이 도트 + 미세 펄스 | sparkle + `animate-pulse` (1.5s) |
| Lv.7 | 무지개 글로우 + 회전 반짝이 | 현재 구현된 glow + conic-gradient 회전 |

### 7-2. 프로필 프레임 확장 (포인트 상점 연계)

포인트 상점에 이미 "프로필 반짝이 프레임" 아이템이 준비 중(comingSoon)으로 있다. 레벨 아이콘 주위에 장착할 수 있는 프레임을 레벨별로 추가 제작하면 시너지가 있다:

- 기본 프레임: 원형 (모든 레벨)
- Lv.4+ 구매 가능: 하트 프레임, 별 프레임, 꽃 프레임
- Lv.6+ 구매 가능: 골드 프레임, 다이아몬드 프레임
- Lv.7 전용: 무지개 프레임 (자동 지급)

## 8. 구현 시 기술 참고사항

### 8-1. 파일 경로 규칙

현재 `constants.ts`에 정의된 경로 패턴:
```
/icons/levels/{petType}_lv{level}.png
```

예시:
- `/icons/levels/cat_lv1.png` (코숏 치즈태비)
- `/icons/levels/other_lv4.png` (왕관앵무)

### 8-2. 연동 코드 (변경 불필요)

현재 코드 구조가 이미 3종 (dog/cat/other) 아이콘을 지원하도록 설계되어 있다:

- `src/config/constants.ts`: `POINT_LEVELS` 배열에 `icons: { dog, cat, other }` 정의 완료
- `src/components/features/points/LevelBadge.tsx`: `petType` prop으로 분기 완료
- `src/components/features/points/PointsBadge.tsx`: `LevelProgress`에 petType 전달 구조 완료

아이콘 PNG 파일만 교체하면 자동으로 적용된다.

### 8-3. petType 결정 로직

사용자의 대표 반려동물 타입에 따라 아이콘 세트가 결정된다:
- 강아지 등록 -> `dog` 아이콘 세트
- 고양이 등록 -> `cat` 아이콘 세트
- 기타 동물 등록 -> `other` 아이콘 세트
- 여러 마리 등록 시 -> 대표(첫 번째) 반려동물 기준

## 9. 예산 추정

### 14개 아이콘 (Cat 7 + Other 7) 커미션 비용

| 방식 | 예상 비용 | 소요 기간 | 장점 | 단점 |
|------|----------|----------|------|------|
| 한국 프리랜서 (크몽 등) | 40만~100만원 | 1~3주 | 한국 감성 이해, 소통 편리 | 상대적 고비용 |
| 해외 프리랜서 (Fiverr) | $70~$350 (10만~50만원) | 3일~2주 | 가성비, 다양한 스타일 선택 | 스타일 차이 가능, 영어 소통 |
| AI 생성 + 후보정 | 5만~20만원 (보정 작업비) | 1~3일 | 빠른 속도, 저비용 | 스타일 일관성 유지 어려움, 세부 표현 한계 |
| 세트 할인 협의 | 위 금액의 70~80% | 동일 | 대량 발주 할인 | 단일 작가에 의존 |

**추천 방식**: 현재 dog 아이콘의 원본 작가에게 동일 스타일로 cat + other 세트를 추가 의뢰하는 것이 스타일 일관성 측면에서 가장 좋다. 원본 작가를 찾을 수 없는 경우, AI 생성(DALL-E 3, Midjourney) + 한국 일러스트레이터의 후보정 조합이 가성비와 품질 모두 확보 가능한 차선책이다.

## 10. 우선순위 및 실행 계획

| 단계 | 작업 | 예상 기간 |
|------|------|----------|
| 1단계 | 본 기획서 확정 및 품종/동물 최종 선정 | 1~2일 |
| 2단계 | 커미션 작가 섭외 + 레퍼런스 전달 | 3~5일 |
| 3단계 | Cat 아이콘 7종 제작 | 1~2주 |
| 4단계 | Other 아이콘 7종 제작 | 1~2주 (3단계와 동시 진행 가능) |
| 5단계 | 아이콘 검수 + 수정 요청 | 2~3일 |
| 6단계 | public/icons/levels/ 폴더에 교체 배포 | 30분 (파일 교체만) |

**참고**: 6단계는 코드 변경이 필요 없다. `constants.ts`의 경로 설정과 `LevelBadge.tsx`의 렌더링 로직이 이미 3종 아이콘을 지원하고 있으므로, PNG 파일만 교체하면 된다.
