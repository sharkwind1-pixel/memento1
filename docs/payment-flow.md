## 결제 경로

사용자가 결제 화면에 도달하는 경로는 아래 3가지입니다.

### 경로 1: AI 펫톡 대화 제한 도달

```
로그인 → 하단 메뉴 [AI펫톡] 탭 클릭 → AI와 대화
→ 무료 대화 횟수(10회) 소진 시 자동으로 프리미엄 구독 모달 표시
→ 베이직 또는 프리미엄 플랜 선택 → [결제 버튼] 클릭
→ 포트원 결제창 → 카드 결제 → 완료
```

- 남은 대화 3회 이하일 때 "프리미엄으로 무제한" 배너도 표시됩니다.
- 대화 횟수 0회 도달 시 프리미엄 구독 모달이 자동으로 열립니다.

### 경로 2: 반려동물 등록 제한 도달

```
로그인 → 하단 메뉴 [내 기록] 탭 클릭
→ [새 반려동물 추가] 버튼 클릭
→ 무료 제한(1마리) 도달 시 프리미엄 구독 모달 표시
→ 베이직 또는 프리미엄 플랜 선택 → [결제 버튼] 클릭
→ 포트원 결제창 → 카드 결제 → 완료
```

- 무료 회원은 반려동물 1마리 등록 후 추가 시 제한됩니다.

### 경로 3: 랜딩 페이지 (비로그인 상태)

```
비로그인 상태로 사이트 접속 → 랜딩 페이지 표시
→ 구독 플랜 비교 섹션 확인
→ [시작하기] 버튼 클릭 → 로그인 유도
→ 로그인 후 위 경로 1 또는 2를 통해 결제 진행
```

## 결제 처리 흐름 (기술)

```
1. 사용자가 플랜 선택 후 [결제 버튼] 클릭
2. 서버에 결제 준비 요청 (POST /api/payments/prepare)
   - paymentId 생성, 금액 서버측 결정 (위변조 방지)
3. 포트원(PortOne) V2 결제창 호출
   - PG사: KG이니시스
   - 결제수단: 신용/체크카드
   - PC: 팝업 방식 / 모바일: 리다이렉트 방식
4. 결제 완료 후 서버에서 검증 (POST /api/payments/complete)
   - 포트원 API를 통해 결제 상태 및 금액 검증
   - 검증 통과 시 프리미엄 활성화
5. 결제 완료 안내 및 프리미엄 기능 즉시 사용 가능
```
