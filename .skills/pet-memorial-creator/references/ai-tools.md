## 영상 생성 AI

### Hailuo (海螺 / MiniMax)
- **URL**: https://hailuoai.video/
- **API**: 있음 (MiniMax API)
- **비용**: ~$0.05-0.1/영상 (5초 기준)
- **퀄리티**: ★★★☆☆
- **장점**:
  - 가격 대비 퀄리티 최고
  - API 안정적
  - 동물 영상 잘 생성
- **단점**:
  - 중국 서비스 (데이터 우려 가능)
  - 가끔 이상한 결과물
- **추천 용도**: MVP, 대량 생성, 가성비 중시

### Kling AI
- **URL**: https://klingai.com/
- **API**: 있음
- **비용**: ~$0.1-0.2/영상
- **퀄리티**: ★★★★☆
- **장점**:
  - 균형 잡힌 가격/퀄리티
  - 동물 표정 잘 표현
  - 모션 자연스러움
- **단점**:
  - Hailuo보다 비쌈
  - 대기 시간 길 수 있음
- **추천 용도**: 메인 프로덕션, 일반 사용자용

### Pika Labs
- **URL**: https://pika.art/
- **API**: 있음
- **비용**: ~$0.1-0.15/영상
- **퀄리티**: ★★★☆☆
- **장점**:
  - 짧은 영상 특화 (3-5초)
  - 빠른 생성
  - 스타일 일관성
- **단점**:
  - 긴 영상 불가
  - 복잡한 동작 어려움
- **추천 용도**: 짧은 클립, SNS용

### Runway Gen-3 Alpha
- **URL**: https://runwayml.com/
- **API**: 있음
- **비용**: ~$0.4-0.8/영상
- **퀄리티**: ★★★★★
- **장점**:
  - 최고 퀄리티
  - 세밀한 컨트롤
  - 안정적인 결과
- **단점**:
  - 비쌈
  - API 비용 부담
- **추천 용도**: 프리미엄 티어, 특별 주문

### Luma Dream Machine
- **URL**: https://lumalabs.ai/
- **API**: 있음
- **비용**: ~$0.15-0.25/영상
- **퀄리티**: ★★★★☆
- **장점**:
  - 자연스러운 물리 표현
  - 빛/그림자 좋음
- **단점**:
  - 동물 특화 아님
  - 가끔 왜곡
- **추천 용도**: 자연 배경 영상

## 이미지 생성 AI

### DALL-E 3 (OpenAI)
- **URL**: https://openai.com/dall-e-3
- **API**: 있음 (OpenAI API)
- **비용**: ~$0.04/이미지 (1024x1024)
- **퀄리티**: ★★★★☆
- **장점**:
  - 프롬프트 이해력 최고
  - 텍스트 렌더링 가능
  - 안정적 API
- **단점**:
  - 스타일 제한적
  - 실사 동물 약간 어색
- **추천 용도**: 일러스트 스타일, 텍스트 포함 이미지

### Midjourney
- **URL**: https://midjourney.com/
- **API**: 비공식 (Discord 기반)
- **비용**: ~$0.05/이미지 (구독 기반)
- **퀄리티**: ★★★★★
- **장점**:
  - 최고 퀄리티
  - 예술적 스타일
  - 동물 이미지 탁월
- **단점**:
  - 공식 API 없음
  - Discord 통해야 함
  - 자동화 어려움
- **추천 용도**: 고퀄 이미지, 마케팅 소재

### Stable Diffusion (자체 호스팅)
- **URL**: https://stability.ai/
- **API**: 자체 구축 또는 Replicate
- **비용**: ~$0.01-0.02/이미지 (자체 호스팅)
- **퀄리티**: ★★★☆☆ ~ ★★★★☆ (모델에 따라)
- **장점**:
  - 최저 비용
  - 완전한 커스터마이징
  - 파인튜닝 가능
- **단점**:
  - 설정 복잡
  - 서버 필요
  - 퀄리티 편차
- **추천 용도**: 대량 생성, 비용 최적화

### Flux (Black Forest Labs)
- **URL**: https://blackforestlabs.ai/
- **API**: Replicate, fal.ai 등
- **비용**: ~$0.03/이미지
- **퀄리티**: ★★★★☆
- **장점**:
  - 빠른 생성
  - 좋은 가성비
  - 실사 스타일 강점
- **단점**:
  - 비교적 신생
  - 일부 스타일 약함
- **추천 용도**: 실사 느낌 이미지

## 얼굴 스왑/편집 AI

### InsightFace (Roop)
- **용도**: 얼굴 스왑
- **비용**: 무료 (자체 호스팅)
- **장점**: 무료, 빠름
- **단점**: 사람 얼굴 특화 (동물은 추가 작업 필요)

### FaceSwap API
- **용도**: 얼굴 교체
- **비용**: ~$0.02/이미지
- **장점**: 쉬운 API
- **단점**: 동물 지원 제한적

### IP-Adapter (Stable Diffusion)
- **용도**: 스타일/특징 전이
- **비용**: 자체 호스팅 시 무료
- **장점**: 동물 특징 유지 가능
- **단점**: 설정 복잡
