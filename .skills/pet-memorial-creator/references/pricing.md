## 원가 계산 공식

```
콘텐츠 1개 원가 = AI API 비용 + 서버/인프라 비용 + 실패 재시도 버퍼(20%)

판매가 = 원가 × 2 (최소 마진 50%)
마케팅 가용 예산 = 판매가 - 원가 - 최소마진
```

### 비용 요소 상세

| 항목 | 설명 | 예상 비용 |
|------|------|----------|
| AI API | 이미지/영상 생성 | $0.04~0.8 |
| 서버 비용 | Vercel/AWS 등 | ~$0.01 |
| 저장소 | Supabase Storage | ~$0.001 |
| 실패 버퍼 | 재생성 대비 20% | 원가 × 0.2 |

## BM 모델 비교

### 모델 1: 건당 과금

| 항목 | 내용 |
|------|------|
| 가격 | 이미지 500원, 영상 2,000원 |
| 장점 | 손해 없음, 명확한 수익 |
| 단점 | 진입장벽, 충동구매 유도 어려움 |
| 적합 | 검증된 퀄리티 후 |

```
월 100건 판매 시:
- 매출: 200,000원
- 원가: ~50,000원
- 순이익: ~150,000원
```

### 모델 2: 프리미엄 구독 포함

| 항목 | 내용 |
|------|------|
| 가격 | 프리미엄 7,900원에 영상 3개/월 포함 |
| 장점 | 기존 BM 통합, 구독 유지 동기 |
| 단점 | 영상 원가 부담 (최대 800원 × 3) |
| 적합 | 저원가 영상 (Hailuo) 사용 시 |

```
구독자 100명, 평균 2개 사용 시:
- 구독 매출: 790,000원
- 영상 원가: ~31,000원 (155원 × 200개)
- 기존 서비스 원가 제외 후에도 충분한 마진
```

### 모델 3: 무료 + 유료 하이브리드

| 항목 | 내용 |
|------|------|
| 가격 | 첫 1개 무료, 이후 1,500원/개 |
| 장점 | 맛보기 → 전환, 바이럴 유도 |
| 단점 | 무료 악용 가능 (계정 생성 반복) |
| 적합 | 초기 사용자 확보 단계 |

```
1000명 가입, 20% 유료 전환 시:
- 무료 원가: 155,000원 (1000 × 155원)
- 유료 매출: 300,000원 (200명 × 1,500원)
- 순이익: 145,000원 (마케팅 전)
```

## 가격 책정 전략

### 멘토 조언 기반 원칙
> "원가가 5천 원이면 1만원 받으면 되고, 원가가 1만원이면 2만원 받으면 되고"

1. **절대 손해 보지 않는 가격**으로 시작
2. **결제 발생 확인** 후 가격 조정
3. **마진 20-30%**는 마케팅 재투자

### 심리적 가격대

| 가격 | 인식 | 전환율 예상 |
|------|------|------------|
| 무료 | 당연히 써봄 | 높음 |
| 500원 | 커피보다 쌈 | 중상 |
| 1,000~2,000원 | 합리적 | 중 |
| 3,000~5,000원 | 고민됨 | 중하 |
| 10,000원+ | 특별한 이유 필요 | 낮음 |

### 가격 테스트 방법
1. A/B 테스트: 1,500원 vs 2,500원
2. 전환율 및 총 매출 비교
3. 최적 가격 도출

## 마케팅 예산 계산

### 공식
```
영상 1개 판매 시 마케팅 가용 예산 = 판매가 - 원가 - 최소마진(30%)

예시 (2,000원 판매, 300원 원가):
가용 예산 = 2,000 - 300 - 600 = 1,100원
→ 영상 1개 팔 때마다 1,100원까지 광고비 쓸 수 있음
```

### CAC (고객획득비용) 기준

| 채널 | 예상 CAC | 비고 |
|------|----------|------|
| 인스타 릴스 (오가닉) | 0원 | 바이럴 필수 |
| 인스타 광고 | 500~2,000원 | 타겟팅 중요 |
| 구글 광고 | 1,000~3,000원 | 검색 의도 높음 |
| 입소문/후기 | 0원 | 가장 효과적 |

### 초기 전략 제안
1. **유료 광고 최소화** - 릴스/숏폼 바이럴 집중
2. **무료 1개 제공** - 결과물 공유 조건
3. **후기 수집** - 감동 스토리 콘텐츠화
4. **광고는 검증 후** - PMF 확인 후 스케일

## 손익분기점 계산

### 가정
- 월 서버비: 50,000원 (Vercel Pro + Supabase)
- 영상 판매가: 2,000원
- 영상 원가: 300원
- 마진: 1,700원/개

```
손익분기 = 고정비 ÷ 개당 마진
         = 50,000 ÷ 1,700
         = 약 30개/월

→ 월 30개 이상 판매 시 흑자
```

### 구독 모델 손익분기
- 월 구독료: 7,900원
- 영상 3개 포함 원가: 465원 (155원 × 3)
- 기타 서비스 원가: 500원 (AI 펫톡 등)
- 마진: 6,935원/구독자

```
서버비 50,000원 ÷ 6,935원 = 약 8명
→ 월 8명 이상 구독 시 흑자
```
