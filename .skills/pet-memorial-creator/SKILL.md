# Pet Memorial Creator

메멘토애니의 핵심 차별화 기능인 **AI 추모 영상/이미지 생성** 개발 가이드.

## 핵심 원칙

1. **손해 안 보는 구조**: 원가 계산 후 마진 확보된 가격 책정
2. **감동 + 바이럴**: 릴스/숏폼에서 공유될 수 있는 감동적 결과물
3. **템플릿 기반**: Epik 스타일로 일관된 퀄리티, 예측 가능한 비용

## 템플릿 컨셉

| 템플릿 | 설명 | 감정 |
|--------|------|------|
| **아기 시절** | 노견/노묘 사진 → 어릴 때 모습 예측, 주인에게 애교 | 그리움, 따뜻함 |
| **천국에서** | 머리 위 링 + 후광, 천국에서 주인을 반기는 모습 | 위로, 평화 |
| **휴양지** | 해변 선베드, 선글라스, 칵테일, 카메라 향해 손 흔들기 | 유쾌함, 안심 |
| **꽃밭** | 꽃밭에서 뛰노는 모습 | 자유, 행복 |
| **무지개다리** | 무지개다리 건너며 뒤돌아보는 모습 | 이별, 희망 |

상세 프롬프트: [references/templates.md](references/templates.md)

## 원가 & 가격 책정

```
영상 1개 원가 = AI API 비용 + 서버 비용 + 버퍼(20%)

예시:
- Hailuo 5초 영상: ~600원
- 권장 판매가: 1,500~3,000원 (마진 50%+)
- 프리미엄 구독 포함 시: 월 3개 제한
```

상세 계산: [references/pricing.md](references/pricing.md)

## 구현 로드맵

1. **Phase 1**: 이미지 생성 (DALL-E/Midjourney)
   - 낮은 원가, 빠른 구현
   - 천국/꽃밭 템플릿

2. **Phase 2**: 영상 생성 (Hailuo/Kling)
   - 바이럴 잠재력 높음
   - 아기시절/휴양지 템플릿

3. **Phase 3**: 고급 기능
   - 얼굴 스왑 (유저 반려동물 → 템플릿)
   - 커스텀 텍스트/음악 추가

## 마케팅 연계

- 무료 1개 제공 → 결과물 공유 유도
- 결과물에 워터마크 or "메멘토애니에서 제작"
- 릴스/숏폼용 9:16 비율 기본 제공
- 감동 스토리텔링 캡션 템플릿 제공
