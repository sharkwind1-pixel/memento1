# 메멘토애니 (Memento Ani) - 프로젝트 전체 요약

## 1. 프로젝트 개요
- **서비스명**: 메멘토애니 (Memento Ani)
- **컨셉**: 반려동물과의 일상부터 이별, 그 이후까지 전 여정을 함께하는 AI 기반 라이프로그 플랫폼
- **대표자**: 안승빈
- **디자인 컨셉**: My Little Puppy 게임 감성, 밝고 파스텔한 천국 느낌
- **컬러**: 하늘색(#38bdf8), 연보라(#a78bfa), 따뜻한 황금(#fbbf24)

---

## 2. 기술 스택
- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS
- **Backend**: Supabase (인증, DB, 스토리지)
- **AI**: OpenAI GPT-4 API
- **Payment**: 포트원 (예정)
- **Deployment**: Vercel (예정)

---

## 3. 핵심 기능
1. **라이프로그 기록**: 사진/글 업로드, 태그 기반 분류, 타임라인 UI
2. **AI 펫톡**: 축적 데이터 기반 프롬프트 엔지니어링, 개인화 대화
3. **AI 추억 영상**: 영상 생성 AI (Runway/Pika 등) 연동 검토 중
4. **디지털 추모관**: 기록 데이터 자동 큐레이션, 감성 UI
5. **치유 커뮤니티**: 영상 공유, UGC 기반 성장 구조
6. **분실동물 찾기**: 지역 기반 기능
